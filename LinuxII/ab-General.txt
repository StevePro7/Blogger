GENERAL commands
04/08/2019

These can be applied anywhere..


hostname		confirm host computer name
whoami			confirm who command runs as

sudo su			switch to administrator

sudo apt-get update	general all-purpose update

sudo apt-get -f install	install outstanding packages

sudo dpkg -i *.deb	install any Debian package

sudo tar tar -xvzf *.zip	Unzip tarball file



sudo add-apt-repository *		Add repository
sudo add-apt-repository --remove *	Remove repository
