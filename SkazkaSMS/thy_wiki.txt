<span style="font-family:verdana;font-size:85%;">

<a href="https://www.smspower.org/Homebrew/Skazka-SMS">Skazka</a> is a Slavic RGP Adventure game originally written in BASIC for Sega SC-3000 and posted to SC-3000 Survivors Facebook <a href="https://www.facebook.com/photo/?fbid=3051652211766546&set=gm.4887403341348676">group</a>. The source <a href="https://bit.ly/3tGV9J3">code</a> is now  ported to <a href="http://en.wikipedia.org/wiki/C_(programming_language)">C</a> using <a href="https://github.com/sverx/devkitSMS">devkitSMS</a> for the <a href="http://en.wikipedia.org/wiki/Master_System">Sega Master System</a>.
