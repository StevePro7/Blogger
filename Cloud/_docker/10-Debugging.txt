Reverse engineer Dockerfile built in GoLang02

Update Dockerfile
docker build -t golang_app:1.0 .
docker images

docker run --rm -d -p 5000:5000 golang_app:1.0
docker ps


Stop all running containers
docker stop $(docker ps -a -q)


Delete all images
docker rmi $(docker images -q)
