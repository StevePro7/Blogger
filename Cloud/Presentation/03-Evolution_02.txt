2. Dawn of Virtualization: 2000s
<br />
  
Introduction of virtualization technologies like those provided by VMware allowed Virtual Machines [VMs] to run a physical server - each VM operating as though it were on own dedicated hardware.
  
<br /><br />
Benefits:
<br />

<tr>
<td>
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Resource Efficiency: multiple VMs could share resources of single server leading to better resource utilization
<br />
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Isolation: VMs provide level of isolation btwn apps = failure of one VM did not affect other VMs
<br />
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Snapshot + Cloning: VM state could be saved + cloned making it easier to replicate environments for scaling
