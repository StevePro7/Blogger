3. Containerization: rise of Docker
<br />
  
Next significant shift = containerization w/ Docker at forefront.  Unlike VMs, containers share the host system OS
run in isolated User space = more lightweight and portable that can startup / shutdown rapidly
  
<br /><br />
Advantages:
<br />
  
<table width="99%" border="1">
<tr>
<td>
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Speed: containers can start almost instantly = applications can be launched and scaled in only a matter of seconds
<br />
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Portability: container images are consistent across environments = "it works on my machine" issues are minimized
<br />
&nbsp;<span style="font-family: 'Wingdings';">•</span>&nbsp;
Density: lightweight nature means many containers can run on host machine = better resource utilization than VMs
</td>
</tr>
