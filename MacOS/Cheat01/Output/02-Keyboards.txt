<u>Keyboards</u><br />
The major differences between Windows and Mac keyboards are the special keys found on either side of the spacebar known as modifier keys and are typically used for keyboard shortcuts.  Here is a quick comparison:
<table width="100%" border="1">
<tr>
<td valign="top">&nbsp;<b>Location</b></td><td valign="top">&nbsp;<b>Windows</b></td><td valign="top">&nbsp;<b>Mac OS</b></td><td valign="top">&nbsp;<b>Symbol</b></td>
</tr>
<tr><td valign="top">&nbsp;Left</td><td valign="top">&nbsp;Ctrl</td><td valign="top">&nbsp;Ctrl</td><td valign="top">&nbsp;@</td></tr>
<tr><td valign="top">&nbsp;Middle</td><td valign="top">&nbsp;WinKey</td><td valign="top">&nbsp;Option</td><td valign="top">&nbsp;~</td></tr>
<tr><td valign="top">&nbsp;Right</td><td valign="top">&nbsp;Alt</td><td valign="top">&nbsp;Command</td><td valign="top">&nbsp;^</td></tr>
<tr><td valign="top">&nbsp;Left</td><td valign="top">&nbsp;Shift</td><td valign="top">&nbsp;Shift</td><td valign="top">&nbsp;$</td></tr>
</table>
<br />
Out-of-the-box, there are some common keyboard shortcuts shared across Windows and Mac, for example:
<br />
<table width="100%" border="1">
<tr><td valign="top">&nbsp;Alt + Tab</td><td valign="top">&nbsp;Toggle between all open applications</td></tr>
<tr><td valign="top">&nbsp;Ctrl + Tab</td><td valign="top">&nbsp;Toggle between all open tabs within an application</td></tr>
</tr>
</table>
<br />
However, for all common Ctrl + type Windows keyboard shortcuts the Mac prefers to use the Command key:
<br />
<table width="100%" border="1">
<tr>
<td valign="top">&nbsp;Ctrl + A</td><td valign="top">&nbsp;Select all</td>
<td valign="top">&nbsp;Ctrl + Q</td><td valign="top">&nbsp;Quit application</td>
</tr>
<tr>
<td valign="top">&nbsp;Ctrl + C</td><td valign="top">&nbsp;Copy item</td>
<td valign="top">&nbsp;Ctrl + S</td><td valign="top">&nbsp;Save item</td>
</tr>  
<tr>
<td valign="top">&nbsp;Ctrl + F</td><td valign="top">&nbsp;Find item</td>
<td valign="top">&nbsp;Ctrl + V</td><td valign="top">&nbsp;Paste item</td>
</tr> 
<tr>
<td valign="top">&nbsp;Ctrl + N</td><td valign="top">&nbsp;New item</td>
<td valign="top">&nbsp;Ctrl + W</td><td valign="top">&nbsp;Close window</td>
</tr> 
<tr>
<td valign="top">&nbsp;Ctrl + O</td><td valign="top">&nbsp;Open item</td>
<td valign="top">&nbsp;Ctrl + X</td><td valign="top">&nbsp;Cut item</td>
</tr> 
<tr>
<td valign="top">&nbsp;Ctrl + P</td><td valign="top">&nbsp;Print item</td>
<td valign="top">&nbsp;Ctrl + Z</td><td valign="top">&nbsp;Undo action</td>
</tr> 
</table>
<br />
Therefore, remap the modifier keys in System Settings | Keyboard | Keyboard Shortcuts... | Modifier Keys:
