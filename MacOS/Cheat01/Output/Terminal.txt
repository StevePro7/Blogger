<u>Terminal</u><br />
The Mac terminal may be <b>zsh</b> by default.  However, change your default shell via Settings... | General | Shell open with: Command path <i>/bin/bash</i> or enter <b>chsh -s /bin/bash</b>.  Confirm BASH is now in Terminal title!
<br /><br />
IMPORTANT: remove silence deprecation warning | Enter the following into Terminal for <b>~/.bash_profile</b>:
<br />

<br />
Customize the Mac terminal to always show full directory path as the cursor similar to the Linux terminal:
<br />
<table width="98%" border="1">
<tr><td valign="top">&nbsp;echo "export PS1='\u@\h:\w$ '" >> ~/.bash_profile</td></tr>
</table>
<br />
Finally, keyboard shortcuts like Ctrl + C or Ctrl + Z may not work to break on Mac thus prefer <b>Ctrl + "."</b>.
