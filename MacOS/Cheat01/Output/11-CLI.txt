<b>Cloud CLIs</b><br />
Download and install Command Line Interfaces for all managed cloud providers e.g.: AWS, Azure, Google:
<a href="https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html">Install</a> the latest version of the <b>AWS</b> CLI.  Download the <a href="https://awscli.amazonaws.com/AWSCLIV2.pkg">macOS pkg</a> file.  Confirm installation <i>aws version</i>.
<a href="https://learn.microsoft.com/en-us/cli/azure/install-azure-cli-macos">Install</a> <b>Azure</b> CLI on macOS using brew <i>brew update && brew install azure-cli</i>.  Confirm install <i>az version</i>.
<a href="https://cloud.google.com/sdk/docs/install-sdk#mac">Install</a> the <b>Google</b> Cloud CLI.  Download the Apple M1 <a href="https://dl.google.com/dl/cloudsdk/channels/rapid/downloads/google-cloud-cli-473.0.0-darwin-arm.tar.gz">package</a>.  Extract and execute <i>install.sh</i> | <i>gcloud init</i>.
