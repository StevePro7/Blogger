Alt + Tab           toggle applications
Ctrl + Tab          toggle tabs e.g. Browser, PyCharm
Ctrl + +Home        start doc
Ctrl + End          end doc
Ctrl + Left         move 1x word back
Ctrl + Right        move 1x word forward
Ctrl + Shift + Left     select 1x word back
Ctrl + Shift +  Right   select 1x word forward
Shift + Home            select all from cursor to start line
Shift + End              select all from cursor to end line
