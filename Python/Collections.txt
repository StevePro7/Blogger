Reference: PythonFundamentals
Python programming language has four collection data types- 
01. list		[]
02. dictionary		{}
03. tuple		()	parentheses
04. sets		{}	curly braces
 
my_tuple	= (1, 2, 3, 4, 5)
my_set 	    	= {1, 2, 3, 4, 5}


01. lists	mutable
[]		square bracekts
mutable sequence of objects


02. dicts	mutable
{}		curly braces
mutable mapping of immutable keys to mutable objects


03.
tuples		immutable
()		parentheses
immutable sequence of arbitrary objects


04. sets	immutable	
{}		curly braces
unordered collection of unique immutable objects


NB:
strings	immutable
