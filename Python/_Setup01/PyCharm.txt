PyCharm shortcuts
08/07/2020


Ctrl + F12
Prompt popup to itemize all methods in class
https://intellij-support.jetbrains.com/hc/en-us/community/posts/206597225-Is-there-a-dropdown-where-i-can-see-all-methods-of-the-class-


Ctrl + Shift + N
Prompt popup to search for text in all files


Ctrl + Shift + I
View definition of function of method in class
https://www.jetbrains.com/help/pycharm/viewing-reference-information.html



Ctrl + Click
Click function to navigate to function definition


Ctrl + Alt + Left arrow
Navigate backward to previous location


Ctrl + Alt + Right arrow
Navigate forward to next location
