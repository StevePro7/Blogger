DOOM Retro is the classic, refined DOOM source port for Windows PC

originally derived from 
Chocolate DOOM
https://github.com/chocolate-doom/chocolate-doom

Written in C
https://en.wikipedia.org/wiki/C_(programming_language)

DOOM Retro source code is maintained in this public Git repository
https://github.com/bradharding/doomretro
https://github.com/bradharding/doomretro/releases/tag/v5.4

and regularly compiled into both 32 and 64-bit Windows apps using
MS Visual Studio Community
https://visualstudio.microsoft.com/vs/community
