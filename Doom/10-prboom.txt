<u>PrBoom</u><br />
PrBoom is a cross-platform version of the classic 3D first-person shooter Doom from id Software.  PrBoom has since been ported to Linux and many other platforms.  PrBoom requires the original IWAD data files.
<br /><br />
Follow this YouTube video to setup PrBoom in VS Code and build and debug step through the source code!
  
<br /><br />
Launch Terminal - 01. install the following packages using sudo apt-get install
<br /><br />
Download prboom-plus-2.5.1.3 from <b><a href="https://sourceforge.net/projects/prboom-plus/files/prboom-plus/2.5.1.3">sourceforge.net</a></b>.  That is the link that seemed to work OK for me.
<br /><br />
Launch Terminal - 02. configure + make
<br /><br />
If you get the following <i>configure: error: *** You must have the SDL libraries installed before you can compile prboom
 *** See http://prboom.sourceforge.net/linux.html</i> then install the following packages:
<br /><br />
Launch Terminal - 03. install the following packages using sudo apt-get install
<br /><br />
Copy all files from the DoomCodeWalkThru Files directory to the local hidden directory <b>~/.prboom-plus</b>
<br /><br />
Launch VS Code.  Create <a href="https://github.com/SteveProXNA/DoomCodeWalkThru/blob/main/Linux/prboom-plus/build.sh">build</a> shell script, hidden <b>.vscode</b> directory with tasks.json and launch.json files
