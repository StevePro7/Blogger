WAD info
14-Jun-2024

WAD
https://doomwiki.org/wiki/WAD

is an acronym for "Where's All the Data"
which is the file format used by Doom and all
Doom engine-based games for storing data

A WAD file consists of a header, a directory and the data "lumps"
https://doomwiki.org/wiki/Lump

that make up the resources stored within the file


//There are 2x types of WAD files

IWAD
https://doomwiki.org/wiki/IWAD
Internal WAD
Core WAD file loaded by the engine that provides all data required to run the game

PWAD
https://doomwiki.org/wiki/PWAD
Patch WAD
Optional file that replaces IWAD data and provides additional data to the engine
