<span style="font-family:verdana;font-size:85%;">
In the previous <a href="https://steveproxna.blogspot.com/2023/09/gbdk-programming-setup.html">post</a>, we checked out GBDK Programming Setup. The <a href="https://github.com/gbdk-2020/gbdk-2020">GBDK</a> is a cross-platform development kit for sm83, z80 and 6502 based gaming consoles including the Nintendo <a href="https://en.wikipedia.org/wiki/Game_Boy">Game Boy</a> and <a href="https://en.wikipedia.org/wiki/Game_Boy_Color">Game Boy Color</a>.
<br /><br />
Using the GBDK, it is now possible to write game code using the <a href="http://en.wikipedia.org/wiki/C_(programming_language)">C language</a> rather than pure <a href="http://en.wikipedia.org/wiki/Zilog_Z80#Z80_assembly_language">Z80 assembly</a>. Therefore, we would like to extend this knowledge and checkout some code samples and homebrew games.
<br />

Let's check it out!

<u>Software</u><br />
Follow all instructions from the previous <a href="https://steveproxna.blogspot.com/2023/09/gbdk-programming-setup.html">post</a>: this documents how to setup all the pre-requisite software.
<br />
Note: ensure you have downloaded and installed the GBDK devkit and Small Device C Compiler [<a href="https://sourceforge.net/p/sdcc/wiki/Home">SDCC</a>].
<br /><br />
<u>Examples</u><br />
