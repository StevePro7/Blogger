<a href="https://en.wikipedia.org/wiki/PyTorch">PyTorch</a> is an open-source machine learning framework widely used for building + training neural networks.  While PyTorch provides the core building blocks for tensors, neural networks, and training loops, most real-world machine learning problems involve domain-specific data—images, audio, graphs + also 3D geometry!

Let's check it out!
<br /><br />
PyTorch ecosystem extends specialized libraries for different domains [vision, audio, 3D, sparse data] go-to choice for everything from prototyping research ideas to deploying production-grade deep learning systems.
  
<br /><br />
PACKAGES<br />
<table width="99%" border="1">
<tr><td valign="top">&nbsp;torch</td><td valign="top">&nbsp;Core PyTorch library providing tensors, autograd and neural network building blocks</td></tr>
<tr><td valign="top">&nbsp;torchaudio</td><td valign="top">&nbsp;Tools + datasets for audio processing, waveform transformations and speech models</td></tr>
<tr><td valign="top">&nbsp;torchvision</td><td valign="top">&nbsp;Computer Vision utilities for datasets: pretrained models and image transformations</td></tr>
<tr><td valign="top">&nbsp;pytorch3d</td><td valign="top">&nbsp;3D Deep Learning library for working with meshes, point clouds rendering pipelines</td></tr>
<tr><td valign="top">&nbsp;torchsparse</td><td valign="top">&nbsp;High-performance sparse tensor library for 3D geometry e.g. LiDAR and voxel grids</td></tr>
<tr><td valign="top">&nbsp;Torch-scatter</td><td valign="top">&nbsp;Optimized operations for scattering and aggregating neural networks tensor values</td></tr>
<tr><td valign="top">&nbsp;Torch-sparse</td><td valign="top">&nbsp;Sparse matrix operations for PyTorch + often used in 3D Geometric Deep Learning</td></tr>
<tr><td valign="top">&nbsp;Torch-cluster</td><td valign="top">&nbsp;Clustering and neighborhood learning query operations e.g. kNN and radius graphs</td></tr>
<tr><td valign="top">&nbsp;Torch-spline-conv&nbsp;</td><td valign="top">&nbsp;Spline-based convolution layers for Geometric Deep Learning on graphs and meshes</td></tr>

<br />

CUDA<br />
<a href="https://en.wikipedia.org/wiki/CUDA">CUDA</a> [Compute Unified Device Architecture] is a proprietary parallel computing platform and API that allows software to use GPUs for accelerated general-purpose processing in high-performance scientific computing.
<br /><br />
Therefore, to leverage PyTorch computations on GPUs one must install nVidia GPU drivers that support your CUDA version, install CUDA Toolkit that includes nvcc compiler with all PyTorch compatible built packages.
<br /><br />  
Development setup example:
Ubuntu 20.04 | Linux kernel 5.15.0-9  | nVidia GPU Driver v535  | CUDA 12.1
<br /><br />

INSTALLATION<br />
Complete nVidia CUDA Toolkit installation guide found <a href="https://docs.nvidia.com/cuda/archive/12.1.0/cuda-installation-guide-linux">here</a> but can be summarized in the following steps: 
<br />  
<table width="99%" border="1">
<tr><td>
<pre style="font-size: 12px;">
&nbsp;&nbsp;uname -r				# 5.15.0-139-generic

</table>

<b>Step #1</b> - <i>Remove partial CUDA installs</i>
<table width="99%" border="1">
<tr><td>
<pre style="font-size: 12px;">
&nbsp;&nbsp;sudo apt remove -y cuda* nvidia-cuda-toolkit
&nbsp;&nbsp;sudo apt autoremove -y
&nbsp;&nbsp;sudo rm -rf /usr/local/cuda*
</pre>
</td></tr>
</table>
<br />
<b>Step #2</b> - <i>Install nVidia Driver 535</i>
<table width="99%" border="1">
<tr><td>

&nbsp;&nbsp;sudo apt install -y nvidia-driver-535

</pre>
