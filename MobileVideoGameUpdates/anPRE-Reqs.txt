Mobile Video Game Upgrades
29/10/2019

PRE-Reqs

As we are targetting both iOS and Android we must upgrade both Window PC and the Mac.
Install the following software

Windows PC [Android]
Install Android Studio
Install Visual Studio 2017
Install Visual Studio 2019
Install "Mobile development with .NET" for Xamarin development.
Install MonoGame for Visual Studio


Mac	[iOS]
Install Xcode 11?
Install Visual Studio for Mac
Install MonoGame for MacOS


This post assumes you already have bundle identifiers setup as the games are already published



Update Android SDK
