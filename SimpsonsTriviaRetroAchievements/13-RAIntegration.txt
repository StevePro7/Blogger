13-RAIntegration<br>
<a href="https://github.com/RetroAchievements/RAIntegration">RAIntegration</a><br />
RAIntegration is the main DLL used for interfacing with <a href="https://retroachievements.org/game/14672">retroachievements.org</a>.  Launch Visual Studio 2022.  Open RA_Integration.sln.  Choose Debug | x64.  Rebuild Solution.  Install CppUnitTest <a href="https://marketplace.visualstudio.com/items?itemName=drleq.CppUnitTestFrameworkAdaptor">Test Adapter</a> to run all Interface + Integration tests: Extensions menu | Manage Extensions | Test Adapter CppUnitTest Framework
<br />
<br />
