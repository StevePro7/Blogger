02-RAaccount<br/>
<u>RetroAchievements</u><br />
<a href="https://emulation.gametechwiki.com/index.php/RetroAchievements">RetroAchievements</a> is an online service which provides users with fan-made achievement sets for many retro gaming platforms such as the Sega Master System.  Navigate to <a href="https://retroachievements.org">RetroAchievements</a> website.  Choose <a href="https://retroachievements.org/createaccount.php">Sign up</a>  Create an account.  Download <a href="https://retroachievements.org/bin/RALibretro-x64.zip">RALibRetro</a> emulator which facilitates achievements integration to retro games!
<br /><br />

05-Implementation<br>
<u>Implementation</u><br />
<a href="https://www.smspower.org/Homebrew/SimpsonsTrivia-SMS">Simpsons Trivia</a> game has 4x difficulty categories: Easy, Normal, Hard and Pro! plus 4x questions rounds: 5, 10, 25, and 50.  Thus 20x achievements setup: Perfect win for 4x categories * 4x rounds plus "Get between 80-100%" for 4x categories.  Shout out to <a href="https://docs.retroachievements.org/developer-docs">Bl4h8L4hBl4h</a> setting up 20x achievements as per <a href="https://docs.retroachievements.org/developer-docs">documentation</a>!
<br /><br />
Also, shout out to BenGhazi explaining how <a href="https://retroachievements.org/downloads">RALibRetro</a> emulator processes achievements e.g. <a href="https://retroachievements.org/achievement/86807">Achievement</a> has 4x important pieces of data stored in RAM: Difficulty [0x00], Selected Rounds [0x05], Correct Answers [0x05] and ScreenID [0x0C].  When all values are aligned on Game Over screen then the achievement is set!
<br />
