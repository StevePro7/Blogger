XX-GitHub<br>
<u>GitHub</u><br />
RetroAchievements <a href="https://github.com/orgs/RetroAchievements/repositories">GitHub</a> includes the following source code repositories for game achievement integration:
<br />
<!-- -->
<table width="99%" border="1">
<tr>
<td valign="top">
&nbsp;<a href="https://github.com/RetroAchievements/rcheevos">rcheevos</a>
</td>
<td valign="top">
&nbsp;Library to parse and evaluate achievements and leaderboards for RetroAchievements
</td>
</tr>
<tr>
<td valign="top">
&nbsp;<a href="https://github.com/RetroAchievements/RAInterface">RAInterface</a>
</td>
<td valign="top">
&nbsp;Enables RetroAchievements emulators to interact with the server via RA_Integration.dll
</td>
</tr>
<tr>
<td valign="top">
&nbsp;<a href="https://github.com/RetroAchievements/RAIntegration">RAIntegration</a>&nbsp;
</td>
<td valign="top">
&nbsp;The DLL responsible to integrate emulators with RetroAchievements.org
</td>
</tr>
<td valign="top">
&nbsp;<a href="https://github.com/RetroAchievements/RALibretro">RALibretro</a>
</td>
<td valign="top">
&nbsp;RALibretro is the multi-emulator used to develop RetroAchievements
</td>
</tr>
</table>
<!-- -->
<br />
<b>Clone</b><br />
On Windows, launch Terminal.  Create C:\GitHub\RetroAchievements directory.  Git clone the following repos:
<br />
<table width="99%" border="1">
<tr>
<td valign="top">
<pre style="font-size: 12px;">
&nbsp;mkdir -p C:\GitHub\RetroAchievements
&nbsp;cd C:\GitHub\RetroAchievements
&nbsp;git clone --quiet https://github.com/RetroAchievements/rcheevos.git
&nbsp;git clone --quiet https://github.com/RetroAchievements/RAInterface.git
&nbsp;git clone --recursive --depth 1 -q https://github.com/RetroAchievements/RAIntegration.git&nbsp;
&nbsp;git clone --recursive --depth 1 -q https://github.com/RetroAchievements/RALibretro.git
</pre>
</td>
</tr>
</table>
<br />
IMPORTANT: use the <b>--quiet</b> [or <b>-q</b>] flag in order to suppress all the superfluous submodule logging output!
