PUSH HL			SP = SP - 2
			POKE SP + 1, H
			POKE SP,     L

POP HL			L = PEEK( SP )
			H = PEEK( SP + 1 )
			SP = SP + 2



Pg.60
JR NC			if carry = 0 then JR x bytes
JR C			if carry = 1 then JR x bytes

JR NZ			last calc is Not Zero then JR x bytes
JR Z			last calc is Zero     then JR x bytes


JP PO			if PV = 0 then JP
JP PE			if PV = 1 then JP

Parity Odd or Even EQU Parity oVerflow

JP PO = JP NV		No oVerflow
JP PE = JP V		oVerflow

e.g.
00 -7F			positive
80 -FF			negative

41h			positive
41h			positive
82h			negative

Overflow occurs if answer gives "wrong" sign
