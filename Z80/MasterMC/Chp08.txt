Pg.99
IX can be used instead of HL	DD
IY				FD
register pair cannot be split	like HL


excep
EX DE, HL
ADC HL
SBC HL

e.g.
LD (HL), 03			36 03
LD (IX+2A), 03			DD 36 2A 03

where 2A			displacement


Pg.100
F				Flags register

Bit 7				Sign flag		S
Bit 6				Zero flag		Z
Bit 5				Not used
Bit 4				Half carry flag		H
Bit 3				Not used
Bit 2				Parity / overflow	PV
Bit 1				Subtract flag		N
Bit 0				Carry flag		C


Half-carry
set to 1 if there is carry from bit 3  to bit 4		single byte
set to 1 if there is carry from bit 11 to bit 12	double byte


IFF1
Interrupt Flip Flop 1
