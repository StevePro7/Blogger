01-Miscellaneous<br />
<b>Terminal</b><br />
Modify Terminal keyboard shortcuts for classic Copy and Paste functionality.  Launch Terminal | Preferences | Shortcuts | Edit.  Double click Shortcut Key next to Copy | enter Ctrl + C.  Repeat for Paste but enter Ctrl + V.

<pre class="csharp" name="code">
Terminal menu | Preferences | Shortcuts | Edit
double click
Copy	Ctrl+C
Paste	Ctrl+V
</pre>

02-VNC<br />
<b>VNC Viewer</b><br />
If you are transitioning from Windows to Linux you may like to remote desktop from Windows to Linux box.
<br /><br />

<i>Windows</i><br />
  
<i>Linux</i><br />
Follow all instructions.  Change Ubuntu desktop manager from GNOME to LightDM.  Install X11VNC as service on Ubuntu.  Configure.  Launch Terminal:
<pre class="csharp" name="code">
sudo apt-get update
sudo apt-get install lightdm
sudo reboot
sudo apt-get install x11vnc
</pre>
