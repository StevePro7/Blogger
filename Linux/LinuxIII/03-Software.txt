03-Software<br />
<b>Software</b><br />
In the previous <a href="https://steveproxna.blogspot.com/2019/10">post</a>, a lot of software was installed from various commands entered in the Terminal window.  Fortunately, a lot of software can be installed directly from <a href="https://wiki.ubuntu.com/SoftwareCenter">Ubuntu</a> Software instead.  Here, an extended list.
<br /><br />
Launch Ubuntu Software.  Search for the following software applications listed below and install accordingly:
